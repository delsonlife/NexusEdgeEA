# NexusEdgeEA

## Expert Advisor MT5 - XAUUSD Intelligent Trading System


NexusEdgeEA est un Expert Advisor développé pour analyser et trader les marchés financiers avec une architecture modulaire avancée.

Le projet combine :

- Smart Money Concepts (SMC)
- Price Action
- Market Structure
- Liquidity Analysis
- Risk Management
- Dynamic Profit Protection


---

# Objectif

Créer un système capable de :

- comprendre le contexte du marché ;
- identifier des opportunités de haute qualité ;
- gérer automatiquement les positions ;
- protéger les profits intelligemment ;
- apprendre des données réelles.
